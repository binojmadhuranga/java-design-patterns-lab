# Software Design Patterns in Java

This repository contains Java practice projects for learning and implementing classic software design patterns.

## Objective

The goal of this repository is to build a strong understanding of design patterns through hands-on examples, small lab projects, and real-world style scenarios.

## Design Pattern Index

The workspace currently includes the following pattern families:

1. Abstract Factory Pattern
2. Adaptor Pattern
3. Composite Pattern
4. Decorator Pattern
5. Observer Pattern
6. Strategy Pattern
7. Template Method

## Pattern Details

### Abstract Factory Pattern

Examples currently included:

* UI Theme Factory
* Vehicle Manufacturing System
* Food Ordering App
* University Course Materials

### Adaptor Pattern

Examples currently included:

* Charger Adapter
* Media Player Adapter

### Composite Pattern

Examples currently included:

* File System Structure
* Shopping Cart / Product Composition

### Decorator Pattern

Examples currently included:

* Coffee customization with milk and sugar
* Cake decoration with cream and chocolate

### Observer Pattern

Examples currently included:

* YouTube subscriber notifications
* Weather station updates

### Strategy Pattern

Current workspace branch under Strategy Pattern:

* Behavioral_Pattern01
* Behavioral_Pattern02
* Template_Method01

### Template Method

Examples currently included:

* Beverage preparation workflow
* Banking transaction workflow

## Learning Approach
